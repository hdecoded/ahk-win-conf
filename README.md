## üîß Setup Instructions

1. **Install AutoHotkey v2.0+** from [https://www.autohotkey.com/](https://www.autohotkey.com/)
2. Clone the repo using `git clone git@github.com:hdecoded/ahk-win-conf.git`
3. Run `productivity.ahk` by double-clicking the file.
4. Optionally, add a shortcut to this script in your Windows startup folder to run it on boot.

---

## üß† 

- Press **CapsLock quickly** to send `Esc`, **hold** it for `Ctrl`
- **Hold Right Ctrl** to activate `Ctrl+Shift+Alt` mode, then use H/J/K/L for arrow navigation.
- Chrome navigation shortcuts (`Ctrl+[` and `Ctrl+]`) only work when Chrome is active.
- Customize app paths in the script if your installations differ from the defaults.
